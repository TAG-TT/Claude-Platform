# Repository Structure Guide

This repository is designed for multi-language AI development.

---

## Top-Level Structure

repo/

agents/ → AI agents
services/ → application services
infrastructure/ → DevOps
configs/ → configuration files
scripts/ → automation scripts
docs/ → documentation

---

## Agents Folder

agents/

Contains autonomous AI agents.

Example:

agents/claude
agents/openai
agents/langchain

---

## Services Folder

services/

Contains application services.

Examples:

api
workers
scheduler

---

## Infrastructure

infrastructure/

Contains:

Docker
Terraform
Kubernetes

---

## Databases

databases/

Contains:

mysql
redis

---

## Data Folder

data/

Used for:

training data
vector embeddings
AI memory
